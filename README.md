# GrapeOS操作系统
## 简介
学习操作系统原理最好的方法是自己写一个简单的操作系统。  
GrapeOS是一个非常简单的x86多任务桌面操作系统，源代码只有四千行，非常适合用来学习操作系统原理。    
视频教程地址：https://www.bilibili.com/video/BV1nv4y1b7Nj/
